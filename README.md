# Materials Shop - Website Bán Nguyên Liệu

Đây là một website e-commerce để bán nguyên liệu xây dựng/công nghiệp.

## 🛠 Tech Stack

- **Frontend**: React + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Authentication**: JWT

## 🚀 Quick Start

### Backend Setup
```bash
cd server
npm install
npm start
```

### Frontend Setup
```bash
cd client
npm install
npm start
```
